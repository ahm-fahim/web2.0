Converting Array to Strings

The JS method toString() converts an array to a string of(comma separated) array values

const fruits = ["Banana", "Orange", "Apple", "Mango"];

document.getElementById("demo").innerHTML = fruits.toString();

Output

Join() Method

The join() also join all array elements into a string.
It behaves just like toString(), but in addintion you can secify the separator.

const fruits = ["Banana", "Orange", "Apple", "Mango"];

document.getElementById("demo").innerHTML = fruits.join(" * ");

Output

Popping & Pushing

When you work with arrays, it is easy to remove elements and add new elements
This is what popping and pushing is:
Popping items out of an array, or pushing items into an array.

pop() method

The pop() method removes the last element from an array:

const fruits = ["Banana", "Orange", "Apple", "Mango"];

fruits.pop();

Output

push() Method

The push() method adds a new element to an array (at the end):

const fruits = ["Banana", "Orange", "Apple", "Mango"];

fruits.push("Kiwi");

Output

Shifting Elements

Shifting is equivalent to popping, but working on the first element instead of the last.

shift() Method

The shift() method removes the first array element and "shifts" all other elements to a lower index.

const fruits = ["Banana", "Orange", "Apple", "Mango"];

fruits.shift();

Output

unshift() Method

The unshift() method adds a new element to an array (at the beginning), and "unshifts" older elements:

const fruits = ["Orange", "Apple", "Kiwi"];

fruits.unshift("Lemon");

Output

Changing Elements

Array elements are accessed using their index number

Array indexes start with 0:
[0] is the first array element
[1] is the second
[2] is the third ...

Example

const fruits = ["Lemon", "Orange", "Apple", "Kiwi"];

fruits[0] = "Passion";

Output

JS array length

The length property provides an easy way to append a new element to an array:

const fruits = ["Passion", "Orange", "Apple", "Kiwi"];

fruits[fruits.length] = "Kiwi";

Output

JS Arrat delete()

Warning !
Array elements can be deleted using the JavaScript operator delete.
Using delete leaves undefined holes in the array.
Use pop() or shift() instead.

const fruits = ["Passion", "Orange", "Apple", "Kiwi","Kiwi"];

delete fruits[0];

Output

Merging (concatenating) Arrays

The concat() method creates a new array by merging (concatenating) existing array

const myGirls = ["Cecilie", "Lone"];

const myBoys = ["Emil", "Tobias", "Linus"];

const myChildren = myGirls.concat(myBoys);

Output

Example (Merging Two Arrays)

The concat() method does not change the existing arrays. It always returns a new array. The concat() method can take any number of array arguments:

Example (Merging Three Arrays)

const arr1 = ["Cecilie", "Lone"];

const arr2 = ["Emil", "Tobias", "Linus"];

const arr3 = ["Robin", "Morgan"];

const myNewArr = arr1.concat(arr2, arr3);

Output

Merging an array with values

const arr4 = ["Emil", "Tobias", "Linus"];

const myNewArr2= arr4.concat("Peter");

Output

Splicing and Slicing Arrays

The splice() method adds new items to an array.

The slice() method slices out a piece of an array.

splice()

const animals = ["dog","cat", "cow"];

animals.splice(2, 0, "goat","megpie");

Output

The first parameter (2) defines the position where new elements should be added (spliced in).
The second parameter (0) defines how many elements should be removed.
The rest of the parameters ("goat" , "megpie") define the new elements to be added.
The splice() method returns an array with the deleted items:

Using splice() to remove elements

with clever parameter setting, you can use splice() to remove elements without leaving "holes" in the array

const animals = ["dog","cat","goat","megpie","cow"];

animals.splice(0,1);

Output

The first parameter (0) defines the position where new elements should be added (spliced in).
The second parameter (1) defines how many elements should be removed.
The rest of the parameters are omitted. No new elements will be added.

JS slice()

The slice() method slices out a piece of an array into a new array.

const animals = ["cat","goat","megpie","cow"];

const delanimels = animals.slice(1);

const delanimels2 = animals.slice(1, 2)

Output

The slice() method creates a new array.
The slice() method does not remove any elements from the source array.
